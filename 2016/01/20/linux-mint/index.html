linux mint 安装使用

万年WIN党的我最近入了Linux这个坑才不到一周，懒惰却又爱折腾，三天内从UBUNTU立马换到了MINT，也是觉得MINT很棒，更适合新手，其实我觉得MINT的概念真的很棒，毕竟系统不是都要那么麻烦，开箱即用的概念是最好的，新手第一次使用这个系统的话起码装完系统就能够上网，听音乐看电影之类的还是很棒的。

细说 Linux-Mint 安装及后续工作

Mint简介

维基百科，自由的百科全书

Linux Mint是一种基于Ubuntu或Debian开发出的Linux操作系统。由Linux Mint Team团队于2006年开始发行。Linux Mint的目标是为家庭用户和企业客户提供一个免费、高效、易用、高雅的桌面操作系统。

放一张我的Linux-Mint图

Mint的安装

系统的下载

Mint 官网:http://www.linuxmint.com/
Mint-基于Ubuntu 下载:http://www.linuxmint.com/download.php
Mint-基于Debian 下载:http://www.linuxmint.com/download_lmde.php

这里我用基于Ununtu的Mint做为教程.

先介绍一下这几个的区别:

Cinnamon: 基于Gnome3桌面进行开发的.
MATE: 基于Gnome2桌面进行开发的.
No codecs: 没有媒体解码器, 有些国家法律严谨你懂得~, 天朝选上面两个版本就可以.
OEM: 预装系统. 自己用就不用选这个.

我使用的是: Cinnamon-64-bit

做U启动盘

Windows: 推荐你使用UltraISO（我是用这个） 或者 rufus:https://rufus.akeo.ie/

Linux: 直接上 dd 命令.

下面用UltraISO制作一个U盘为例子

准备容量比iso镜像文件大的u盘一个，插入电脑的usb接口。启动ultraiso，在左下本地目录一栏中打开iso所在文件夹，在右侧双击打开linuxmint-17.3-Cinnamon-64-bit.iso文件。

点击菜单栏中的启动-写入硬盘映像，在硬盘驱动器中选你的u盘，点写入即可。

准备安装linuxmint的分区。在磁盘管理程序中为linux腾出一个分区来，操作比较傻瓜式，不过注意备份数据，不当的操作会导致数据损失。

重启电脑，通电后修改bios设置，若不成功，多重启几次。U盘启动后，会有如下图界面。按回车即可。

开始安装

之后进入mint的livecd系统。

如图，双击光盘图标运行安装程序。

点击: Install Linux Mint 进行安装

在左侧选择中文简体，点击继续。

注意:

一定要选自己创建调整分区的选项，否则会清除掉windows。

如果你使用Oracle VM进行安装的时候可以选择Ubuntu 切忌磁盘大小不能使用默认的,默认的8GB大小不够, 最好给到20GB动态磁盘反正不会占用这么多.

关于磁盘这里你要多留意一下, 你要装双系统的话,自己看着勾选, 这里就不多说了.

下面的安装启动引导器的设备中一定选你的硬盘，不要错选成u盘。选中为linux腾出来的分区点删除，此时该分区变为空闲。

选中空闲，点击添加。

下面以500G硬盘为例子，其他大小你们可以按比例缩小。

选择主分区，起始位置，/ ，ext4 分配100GB

再把剩余的空闲点击添加

选择主分区，起始位置，交换空间 分配2GB（因为我的内存是8GB所以交换空间弄的不大）

选择主分区，起始位置，/ home，ext4 分配100GB

剩余的选择 主分区， 起始位置，/自定义

点击现在安装，则出现下部的进度条：

这几步可以直接使用默认设置自己设置自己的账户就可以了.

安装过程中可以看到各种特色介绍, 如果安装的时候就联网了. 可以在联网下载的时候进行跳过. 因为源的问题速度很慢, 令人发指..

安装完成后，在弹出的对话框上点现在重启。记得拔下u盘啊，否则就又进入u盘上的系统了。重启后会有下图界面：

之后输入你的用户名, 密码, 就可以登录系统了.

到此系统的安装就完毕了.

安装完系统后.

第一件事就是改变软件源和更新你的系统：

首先就要做的也是最重要的就是修改软件源并且增加 Canonical Parters （canonial合作伙伴）。

这将会增加你软件库软件和程序的数量并且你可以更加容易的安装它们无论是通过ubuntu中心或者是命令行。

毕竟还是用国内的镜像更新速度快. 中间会进行一次密码确认

主要和基础两个镜像, 点击后会有测速. 自己选择最快的即可~

选择好后要点击上方的更新缓存.

如果中间出现什么错误提示,可以换一下镜像试试.

缓存更新完毕后,就可以关闭了。

打开 Unity Dash （按Windows键或者是super(manbook的电脑)），搜索“软件更新”。

在另一个标签里面打开它，确保 Canonical Parters已经检查过了

这么做以后将会更新你的软件包信息，这将会花费一些时间，一旦你这个做，打开终端（ctrl+alt+t）敲入以下的命令

sudo apt-get update&&sudo apt-get upgrade

上面的命令将会更新你的系统。

更新完毕后最好重启一下系统在进行后续的操作.

科学上网

这里推荐使用COW: https://github.com/cyfdecyf/cow
感谢作者: cyfdecyf
虽然安装过程GitHub上有.这里在提一下.
这里推荐的是使用 SS 进行科学上网.

安装

sudo curl -L git.io/cow | bash

#执行这条命令进行安装和更新 中间会问你安装到哪个目录下, 这里建议安装到 /usr/local/bin 这个目录里.

编辑配置文件

sudo gedit  ~/.cow/rc  # 编辑配置文件.

关于配置文件的细节我就不细说了, 里面都是中文

listen = http://127.0.0.1:7777 # 这行里面有如果想改端口直接修改这行即可.

下方是配置SS帐号的写法

proxy = ss://加密方法:ss密码@ss帐号:端口

该完后保存退出

运行

cow & # 如果不行的话可以 sudo 提权

这种方式的运行 不可以关闭当前的shell窗口

管理脚本

作者提供的启动脚本: https://github.com/cyfdecyf/cow/blob/master/doc/init.d/cow
复制下来之后创建文件粘贴进去

sudo gedit /etc/init.d/COW # 创建cow文件 并把启动脚本全部复制到这里

提权 

来这个地方/etc/init.d/COW

对COW打开属性，设置权限，勾起允许以程序执行文件

需要更改的有一下几个地方.

BIN=/usr/local/bin/cow # 程序安装的目录

USER=xxxx # 你的用户名

GROUP=xxxx # 你的用户组 用户组和用户名一致

#该完后保存退出

开机自启动

sudo update-rc.d COW defaults

使用

火狐浏览器

打开后在网址栏输入: about:preferences
Advanced -> Network -> coeeection -> settings -> Automatic proxy configuration URL: 运行COW提示的链接
eg: http://127.0.0.1:7777/pac 17.png
保存后测试看看.

Google Chrome

点击右下方 Meun -> 控制中心 -> 互联网和网络 -> 网络代理 -> 自动配置代理
和火狐的配置一样. http://127.0.0.1:7777/pac

学Linux半途而废不如从不上路

有问题问是个好习惯，但是一遇到问题就问或简单的问题也问是个坏习惯。

要学到东西，自己看书还有去专业的论坛才是王道。然后自己懂了些东西，再一边学习，一边教别人。

学Linux是件麻烦事。

作为一名大学生，在沉重的学业下，学这个确实很花时间。眼看着同学用QQ，用Photoshop，自己没有。

眼看着同学华丽的UI，你的却是丑陋的。同学打开热点，你却折腾半天也不会。动摇是很正常的。

但是只有在一段时间的努力学习后，你才会发现原来你已经走了这么远了。你可以离开盗版了，可以离开windows了。

半途而废不如从不上路。

如果你们不能坚持，Linux永远不能成长。我们永远要被windows绑架。

诚然，windows也好，方便易用。但是你们认为几百人民币的价格是合理的吗？错了，如果没有垄断，系统不值这个价。

希望对某些人有用。
